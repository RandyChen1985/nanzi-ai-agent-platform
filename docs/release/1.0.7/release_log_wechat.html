🚀 NanZi AI 开源智能体平台 · v1.0.7 重磅发布

品牌统一 · ChatBI可信分析 · 黄金报表订阅 · 个人技能闭环

项目声明

本项目为个人开源，供自由学习交流，遵循 MIT 开源协议，可自由分发。

「NanZi」来自我一直使用的网名（南孜），取「孜孜不倦」之意，寓意 AI 持续学习与进化。

GitHub Repository： RandyChen1985/nanzi-ai-agent-platform

v1.0.7 是一次以“NanZi品牌统一、ChatBI 可信分析、黄金报表订阅、Grounding 门禁、个人技能与场景工作台”为核心的里程碑大版本迭代。平台完成「 Yunshu」→「合思/ NanZi」全站更名；ChatBI 补齐非数据分流、连续分析与可信依据；黄金报表支持订阅、智能简报与告警推送；引入 Evidence Grounding 与 MCP 会话恢复；并打通个人技能创建-编辑-挂载闭环与场景模板交付。

本次变更范围自 cda13f99（含）至 3aee21bb，共 101 个提交，涉及 535 个文件、约 47,974 行新增代码。

🚀 Key Features

01 🏷️ 品牌统一：合思/ NanZi

●全站文案更名： Yunshu → NanZi，登录页、侧栏、嵌入助手与文档同步。

●仓库对齐：GitHub 仓库更名为 nanzi-ai-agent-platform，README 与项目声明更新。

02 📊 ChatBI 分诊、连续分析与可信交付

●非数据分流：非数据意图结构化澄清，放宽软性拦截，禁止把 dataset 当库名。

●连续分析升级：分诊与连续分析交付增强，DATA_QUERY 无条件委派。

●可信依据与简报：追问分析 + 证据化业务简报落库（chatbi_briefs）。

03 📈 黄金报表订阅、简报与告警

●订阅与运行历史：定时调度、运行快照、通知中心与站内信箱。

●智能简报推送：AI 分析简报经钉钉 / 企微 / 邮件 / 站内信投递并审计。

●告警条件：版本化告警规则与触发证据，完善订阅通知体验。

04 🛡️ Grounding 门禁与 MCP 稳定性

●Evidence Grounding：统一 GroundingService；Intent-First、Session 门控、Final SQL 角色。

●软风险提示：未接地事实 PASS_WITH_WARNING；MCP external_tool 凭证校验。

●会话恢复：Direct HTTP SSE 解析修复；失效会话自动恢复与重连。

05 🧩 个人技能闭环与 Skills 工作台

●个人技能隔离：独立管理、启用/禁用、排序与作用域修复。

●创建-编辑-挂载：个人中心「我的技能」、创建引导横幅、个人 SKILL.md 预加载。

●导入与白名单：压缩包导入、编辑器升级；智能体版本可自定义公共 Skills 白名单。

06 🧰 场景模板、工作台与摸排画像

●场景交付闭环：场景包市场、资源绑定与交付实例落库；个人业务工作台上线。

●库表摸排画像：中断/分页/增量全量摸排，置信度过滤，直导元数据导入。

●管理端体验：版本配置全屏向导、聊天 Markdown 历史、移动端工具栏与多页视觉对齐。
